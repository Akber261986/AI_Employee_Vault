# AI Employee - Update Dashboard Skill

This skill updates the Dashboard.md file with current status and activity.

## Usage
```
/update-dashboard
```

## What it does
1. Counts pending tasks in /Needs_Action
2. Counts completed tasks in /Done (today)
3. Updates Dashboard.md with current stats
4. Adds recent activity entries
5. Updates timestamp

## When to use
- After processing tasks
- At the start of each day
- When significant events occur
- On manual request
