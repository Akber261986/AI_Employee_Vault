# AI Employee - Process Tasks Skill

This skill processes files from the Needs_Action folder, analyzes them, and takes appropriate action.

## Usage
```
/process-tasks
```

## What it does
1. Scans the /Needs_Action folder for pending tasks
2. Reads and analyzes each task file
3. Creates a plan for complex tasks
4. Executes simple tasks immediately
5. Updates Dashboard.md with progress
6. Moves completed tasks to /Done folder

## Examples
- Process dropped files from Inbox
- Analyze task requests
- Create action plans
- Update status tracking
