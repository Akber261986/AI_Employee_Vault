# AI Employee - Process Email Tasks Skill

This skill processes email tasks from Gmail watcher and creates draft replies.

## Usage
```
/process-emails
```

## What it does
1. Scans Needs_Action for email tasks
2. Reads email content and context
3. Drafts appropriate replies
4. Creates approval requests for sending
5. Updates Dashboard with email activity

## When to use
- After Gmail watcher detects new emails
- To handle email backlog
- For automated email triage
